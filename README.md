## Development Setup

Install the following:

- Node and npm
- Ruby and Bundler
- Gulp `npm install -g gulp`

Clone the repository and run the following command to install all required
dependencies.

```shell
$ npm install
```

### Serving

```shell
$ gulp serve
```

### Other Commands

- `gulp`
- `gulp clean`
- `gulp stage`
- `gulp prod`
